# FaceApiDemoV2
Dự án chủ yếu bao gồm hai mô-đun chức năng: **FireflyApi** và **FaceApi**.

## FireflyApi
FireflyApi bao gồm các chức năng liên quan như **IDCard (thẻ căn cước/CMND)**, **ICCard**, **mã QR**,... FireflyApi có thể linh hoạt phát triển ứng dụng phần mềm tầng trên dựa trên nhu cầu nghiệp vụ thực tế, kết hợp với các API mà SDK cung cấp.

## FaceApi
FaceApi là bộ API lập trình cho nhận diện khuôn mặt, bao gồm các chức năng như **phát hiện khuôn mặt**, **nhận diện khuôn mặt**, **phát hiện thuộc tính khuôn mặt**.
