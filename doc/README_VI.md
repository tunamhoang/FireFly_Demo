# Tài liệu tiếng Việt trong thư mục `doc`

Tệp này cung cấp bản mô tả tiếng Việt cho các tài liệu gốc tiếng Trung trong thư mục `doc`.

## Danh sách tài liệu đã được Việt hóa thông tin

| Tệp gốc | Tên tiếng Việt tương ứng | Mô tả |
| --- | --- | --- |
| `FaceEngine说明文档V2.pdf` | `Tài_liệu_FaceEngine_V2.pdf` | Tài liệu hướng dẫn FaceEngine phiên bản V2 |
| `FaceEngine集成说明截图.png` | `Ảnh_chụp_hướng_dẫn_tích_hợp_FaceEngine.png` | Ảnh chụp minh họa các bước tích hợp FaceEngine |
| `FireflyApi说明文档V2.pdf` | `Tài_liệu_FireflyApi_V2.pdf` | Tài liệu hướng dẫn FireflyApi phiên bản V2 |

## Ghi chú
- Các file nhị phân gốc (`.pdf`, `.png`, `.apk`, `.aar`) vẫn được giữ nguyên tên để tránh ảnh hưởng liên kết/tích hợp hiện tại.
- Bản Việt hóa ở đây là phần mô tả và định danh tài liệu bằng tiếng Việt để dễ tra cứu trong quá trình sử dụng.
- Nếu cần Việt hóa **toàn bộ nội dung bên trong PDF/ảnh**, cần có bước OCR/trích xuất nội dung rồi biên dịch lại thành tài liệu mới.
